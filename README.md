# FLUX.2 —— AI 绘图的新王者

![Flux.2 Hero Image](https://github.com/black-forest-labs/flux/raw/main/assets/flux-logo.png)

> 媲美 Midjourney v6 的极致画质，终于“听得懂人话”的生成式 AI

## 🌟 项目简介

FLUX.2 不仅仅是一个绘图模型，它是**视觉智能**的进化。它告别了传统的“抽卡”模式，用大语言模型（LLM）级的理解能力，精准还原你脑海中的每一个细节。

## 🚀 核心优势

### 01 视觉智能：它懂物理，也懂光影
FLUX.2 经过深度训练，理解真实世界的物理规律、透视关系和光照原理。
*   **不再有违和感**：无论是复杂的反射、折射，还是多光源下的阴影表现，Flux.2 都能处理得自然逼真。
*   **400 万像素直出**：支持生成高达 4MP 的超高清图像，细节经得起无限放大。

### 02 生产力革命：文字渲染与多图参考
*   **完美文字渲染**：准确生成你指定的单词和句子，不再是乱码。
*   **多图参考 (Multi-Reference)**：支持同时参考多张图片，保持角色一致性。

### 03 精准指令跟随
基于先进的架构，Flux.2 能像 ChatGPT 一样理解复杂的长难句。
*   **复杂场景轻松驾驭**：精准落实画面中每个角落的细节。
*   **手部与肢体修复**：大幅降低手指和肢体的错误率。

### 04 强大的开源生态
*   **社区驱动**：Civitai 和 Hugging Face 上有大量微调模型（LoRA）和 ComfyUI 工作流。
*   **可定制性**：满足从二次元到写实摄影风的各种需求。

## 💻 硬件建议

FLUX.2 的强大伴随着对硬件的极致考验：

*   **极致体验 (完整版 32B 参数)**：需要约 **64GB+ 显存**。
*   **主流推荐 (4-bit 量化版)**：
    *   推荐使用 `diffusers/FLUX.2-dev-bnb-4bit` 版本。
    *   **显卡要求**：**RTX 3090 / 4090 (24GB VRAM)**。
